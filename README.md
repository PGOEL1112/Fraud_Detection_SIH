# 🌿 Intelligent Fraud Detection System (ML + Full Stack)

## 🚀 Overview

This project is a **machine learning-powered fraud detection system** designed to identify suspicious transactions in herbal product supply chains.
It combines **ML models + rule-based logic + full-stack web interface** for real-time and bulk fraud analysis.

---

## 🔥 Features

* 🔍 Real-time fraud prediction using input form
* 📂 CSV upload for bulk transaction analysis
* 📊 Interactive dashboard with transaction history and charts
* 🤖 Machine Learning models (Random Forest + XGBoost)
* ⚡ Hybrid logic (ML + rule-based validation)
* 🔁 Dynamic model retraining with new datasets
* 🧠 Fraud explanation system (reasons for fraud detection)

---

## 🛠 Tech Stack

* **Backend:** Flask (Python)
* **Machine Learning:** Scikit-learn, XGBoost
* **Frontend:** HTML, CSS, JavaScript
* **Data Processing:** Pandas, NumPy
* **Model Persistence:** Joblib

---

## 🧪 How It Works

1. Dataset is generated using a synthetic data generator
2. Data is preprocessed and encoded
3. ML models are trained and saved
4. User inputs or CSV data is processed
5. System predicts whether transaction is **Fraud or Safe**
6. Displays result with explanation

---

## 📈 Model Details

* Random Forest Classifier
* XGBoost Classifier
* Probabilistic dataset labeling for realistic fraud patterns
* Feature engineering and validation included
